# Range Savings Calculator (Local Preview)

This folder contains a React-based calculator component you can run locally with Vite.

## Prerequisites

- Node.js 18+ (Node 20 works)
- npm

## Quick Start

1. Open a terminal in this folder:
   - `/Users/leswins/Justforfun/Range Energy/Calculator`
2. Install dependencies:
   - `npm install`
3. Start the dev server:
   - `npm run dev -- --host localhost --port 5173`
4. Open in browser:
   - `http://localhost:5173/`

## Daily Use (After First Setup)

- Start:
  - `npm run dev -- --host localhost --port 5173`
- View:
  - `http://localhost:5173/`
- Stop:
  - Press `Ctrl + C` in the terminal running the server

## Build for Production (Optional)

- Create production build:
  - `npm run build`
- Preview production build locally:
  - `npm run preview`

## Project Files

- `range-savings-calculator.jsx` - main calculator component
- `main.jsx` - app entry that renders the calculator
- `index.html` - HTML shell
- `vite.config.js` - Vite config
- `package.json` - scripts and dependencies
